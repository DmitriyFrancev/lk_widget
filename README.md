# Проект lk_widget #

## Описание ##
Проект содержит функционал личного кабинета клиента банка.

## Установка ##
Для установки проекта необходимо:
1. Клонировать репозиторий: git clone git@github.com:DmitriyFrancev/lk_widget.git
2. Установить зависимости: poetry install

## Использование ##
Примеры использования функций проекта:
- `get_mask_card_number("7000792289606361")` - вывод: `7000 79** **** 6361`
- `get_mask_account("73654108430135874305")` - вывод: `**4305`
- `mask_account_card('Visa Classic 6831982476737658')` - вывод: `Visa Classic 6831 98** **** 7658`
- `get_date('2024-03-11T02:26:18.671407')` - вывод: `11.03.2024`
